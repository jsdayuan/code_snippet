## 4. 部署方案

### 4.1 快速部署选项

| 平台 | 特点 | 部署命令 |
|------|------|----------|
| **Vercel** | 最快部署，自动SSL | `vercel` |
| **Netlify** | 免费自定义域名 | `netlify deploy --prod` |
| **GitHub Pages** | 免费托管 | `npm run deploy` |

### 4.2 构建命令

```bash
cd frontend
npm run build
```

构建产物位于 `frontend/dist` 目录，可直接部署到任意静态文件服务器。
